考研《数据结构》第一章：绪论（二）算法与算法评价

算法基本概念

程序 = 数据结构 + 算法

  • 数据结构:程序要处理的信息
  • 算法:处理信息的步骤

算法五大特性(必须全部具备)

  1. 有穷性:必须在有穷步后结束,每步在有穷时间内完成
    • 算法必须有穷,程序可以无穷(如操作系统)
  2. 确定性:相同输入产生相同输出,无二义性
  3. 可行性:操作可通过基本运算有限次实现
  4. 输入:0个或多个输入
  5. 输出:1个或多个输出

好算法的标准

  1. 正确性:正确解决问题
  2. 可读性:逻辑清晰易懂
  3. 健壮性:能处理非法输入
  4. 高效率:时间复杂度低
  5. 低存储:空间复杂度低

时间复杂度

计算规则(大O记法)

加法规则(并列循环):$O(f(n)) + O(g(n)) = O(\max(f(n), g(n)))$

  • 例:先后执行一重循环和二重循环,总复杂度为$O(n^2)$

乘法规则(嵌套循环):$O(f(n)) \times O(g(n)) = O(f(n) \times g(n))$

  • 例:外层循环n次,内层循环$\log_2 n$次,总复杂度$O(n\log_2 n)$

常见复杂度排序(必背)

口诀:“常对幂指阶”

$$O(1) < O(\log_2 n) < O(n) < O(n \log_2 n) < O(n^2) < O(n^3) < O(2^n) < O(n!) < O(n^n)$$

三种情况

  • 最坏时间复杂度:考研通常只关注最坏情况,代表算法底线
  • 平均时间复杂度:所有输入等概率出现的期望运行时间

对数阶推导

设执行x次,则$2^x > n$,解得$x = \log_2 n$,复杂度为$O(\log_2 n)$

空间复杂度

原地工作

只声明有限个局部变量,与问题规模n无关,空间复杂度为$O(1)$

递归空间开销

递归调用时系统开辟函数调用栈,每次递归压入新栈帧

结论:递归算法空间复杂度 = 递归深度

例:求n阶乘递归深度为n,空间复杂度$O(n)$

答案与解析

选择题

01. B 算法五大特性:有穷性、确定性、可行性、输入、输出

02. C 算法必须具备五个特性

03. C 大O符号表示执行时间与$n^2$成正比

04. B 辅助空间大小与问题规模n无关

05. D 对数阶最小

06. D i每次乘2,执行$\log_2 n$次

07. D $i^3 \le n$,即$i \le n^{1/3}$

08. D 等差数列求和$\sum_{i=2}^{n-1} (i-1) = \frac{(n-2)(n-1)}{2}$,最高阶$n^2$

09. A 最坏情况执行嵌套循环$n^2$

10. A $\sum_{i=1}^{n} 2i = n(n+1)$

11. B 主定理:$T(n) = 2T(n/2) + n$,共$\log_2 n$层,每层$O(n)$

12. A x每次乘2,执行$\log_2 n$次

13. B 递归深度n,每次常数操作

14. C 外层$\log_2 n$次,内层n次

15. B 循环k次后$sum = \frac{k(k+1)}{2} \ge n$,解得$k \approx \sqrt{n}$

16. B $(x+1)^2 > n$,即$x \approx \sqrt{n}$

17. B 总次数$1 + 2 + 4 + \dots + 2^k = 2^{k+1} - 1 < 2n$,为$O(n)$

18. B $\sum_{i=1}^{\sqrt{n}} i = \frac{\sqrt{n}(\sqrt{n}+1)}{2} = \frac{n}{2} + \frac{\sqrt{n}}{2}$,为$O(n)$

综合题

① $O(n)$ - 循环n次,每次常数操作

② $O(\sqrt{n})$ - $(y+1)^2 > n$时结束

③ $O(m \times n)$ - 双层独立嵌套循环

进阶:复杂度变形

若将第18题内层改为j<=i*i:

推导过程

外层循环:i从1到$\sqrt{n}$

内层循环:对每个i执行$i^2$次

总次数:$T(n) = 1^2 + 2^2 + 3^2 + \dots + (\sqrt{n})^2$

使用平方和公式:$\sum_{k=1}^{m} k^2 = \frac{m(m+1)(2m+1)}{6}$

代入$m = \sqrt{n}$:

$$T(n) \approx \frac{\sqrt{n}(\sqrt{n}+1)(2\sqrt{n}+1)}{6}$$

最高阶项:$\sqrt{n} \times \sqrt{n} \times 2\sqrt{n} = 2n^{3/2}$

答案:$O(n\sqrt{n})$ 或 $O(n^{1.5})$
